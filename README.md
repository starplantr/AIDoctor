# AIDoctor

👥 참여자
* 유진식 : AI
* 하윤호 : frontend
* 구혁모 : server
* 박선균 : backend

🛠️ Feature
---
다음의 기능들을 담은 서비스를 구현했습니다.
* 의료 챗봇 서비스
* 의료 챗봇 서비스는 chatgpt를 프롬프트 엔지니어링으로 구현
* 소화기내과 질병진단 서비스
* 질병진단 서비스는 gpt4로 생성한 데이터로 학습한 BERT기반의 모델로 구현
* 로그인 및 회원별 데이터베이스 관리
